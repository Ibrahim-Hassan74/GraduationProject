# Code of Conduct

As the creators of the Wasla platform, we want this project to be a welcoming and respectful environment for anyone interested in exploring the codebase, learning, or contributing.

## Our Standards

When interacting with this repository—whether through issues, pull requests, or discussions—please adhere to the following basic expectations:

* **Be respectful and professional:** Treat everyone with kindness and empathy. Constructive feedback is always welcome; personal attacks are not.
* **Communicate constructively:** Focus on the code and the problem at hand. Be open to differing viewpoints and experiences.
* **Keep it welcoming:** This is a portfolio/graduation project intended for learning and collaboration. Trolling or exclusionary language of any kind will not be tolerated.

## Unacceptable Behavior

The following behaviors are considered unacceptable and will not be tolerated in this project's community:
- Insulting, derogatory, or discriminatory comments
- Spamming issues or pull requests
- Any conduct that could reasonably be considered inappropriate in a professional setting

## Reporting Issues

If you experience or witness unacceptable behavior, or if you have any concerns regarding conduct in this repository, please reach out directly.

You can report issues by emailing us at: **[ibrahimhassan.dev1@gmail.com](mailto:ibrahimhassan.dev1@gmail.com)**

All reports will be reviewed promptly and kept confidential.

## Enforcement

As the maintainers of this project, we reserve the right to close issues, reject pull requests, or block users who violate these guidelines or disrupt the project environment.
